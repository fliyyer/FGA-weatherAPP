# Project Submission - Project Weather-APP

Clone repository

```bash
  git clone https://github.com/fliyyer/bootcamp-noteAPP.git
```

Install packages

```bash
  npm install
```

Running Expo Project

```bash
  npm run start
```

Endpoint WeatherApi

```bash
  https://api.openweathermap.org/data/2.5/weather?lat={lat}&lon={lon}&appid={YOUR_API_KEY}
```
